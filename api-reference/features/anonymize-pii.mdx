Glaider allows you to anonymize personally identifiable information (PII) in text data before processing it with AI models or storing it.

## Endpoint

```
POST https://api.glaider.it/v1/anonymize-pii
```

## Headers

| Header        | Value               |
|---------------|---------------------|
| Authorization | Bearer YOUR_API_KEY |
| Content-Type  | application/json    |

## Request Body

| Parameter | Type   | Required | Description                |
|-----------|--------|----------|----------------------------|
| prompt    | string | Yes      | The input text to anonymize|

## Example Request

```json
{
  "prompt": "i am Lorenzo and I live on the Moon"
}
```

## Response

| Status Code | Description            |
|-------------|------------------------|
| 200         | Successful response    |
| 400         | Bad request            |
| 401         | Unauthorized           |
| 429         | Too many requests      |
| 500         | Internal server error  |

### Success Response Body

```json
{
  "anonymized_text": "i am [PER_0] and I live on the Moon",
  "entities": {
    "[PER_0]": "Lorenzo"
  }
}
```

## Supported Entity Types

The API automatically detects and anonymizes the following types of sensitive information:

- Personal names
- Location names
- Organization names
- Email addresses
- Private keys
- IP addresses
- Access tokens
- API keys
- Credit card numbers

## Code Examples

### Python

```python
import requests

url = "https://api.glaider.it/v1/anonymize-pii"
api_key = "YOUR_API_KEY"

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

data = {
    "prompt": "i am Lorenzo and I live on the Moon"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())
```

### Go

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    url := "https://api.glaider.it/v1/anonymize-pii"
    data := map[string]interface{}{
        "prompt": "i am Lorenzo and I live on the Moon"
    }
    jsonData, _ := json.Marshal(data)

    req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonData))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    fmt.Println("Response Status:", resp.Status)
    // Read and print response body here
}
```

### Java

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class Main {
    public static void main(String[] args) throws Exception {
        String url = "https://api.glaider.it/v1/anonymize-pii";
        String jsonBody = "{\"prompt\":\"i am Lorenzo and I live on the Moon\"}";

        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(jsonBody))
            .build();

        HttpClient client = HttpClient.newHttpClient();
        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());

        System.out.println("Status code: " + response.statusCode());
        System.out.println("Response body: " + response.body());
    }
}
```
